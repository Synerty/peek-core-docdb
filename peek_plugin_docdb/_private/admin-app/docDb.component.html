Document DB

This plugin stores JSON like 'Documents', and supports offline caching.

The documents are used to store information about objects, for example, A piece of equipment might have a document that looks like this:

                {
                    key: "ABC123",
                    document: {
                    {
                        alias:"A12345678COMP",
                        name: "This is a circuit breaker ABC123",
                        rating: "11kV"
                    }
                }
